Reports unresolved file references in Vento template files.

This inspection checks layout, include, and import elements to ensure that the referenced files exist in the project.

Examples:


  {{ layout "templates/base.vto" }}  // Warning if file doesn't exist
  {{ include "partials/header.vto" }}  // Warning if file doesn't exist
  {{ import { foo } from "utils.js" }}  // Warning if file doesn't exist
